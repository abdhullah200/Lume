## 📖 About
 
Lume is a social media application focused on sharing moments and connecting with others. Post stories, interact with content through likes, and build a community — all in a clean, intuitive experience.
 
> ⚠️ **Early Stage** — Lume is currently in active development. Features are being built out and things may change frequently.
 
---
 
## ✨ Features
 
- 📖 **Stories** — Post short-lived stories for your followers to see and like
- ❤️ **Likes** — Like posts and stories to engage with content
- 👤 **User Profiles** — Personal profiles to represent yourself on the platform
- 👥 **Follow System** — Follow other users and build your network
> 💬 **Direct Messages** — Not yet available. May be added in a future release.

## 🚀 Getting Started
 
### Prerequisites
 
- [.NET SDK](https://dotnet.microsoft.com/download) (latest)
- [SQL Server](https://www.microsoft.com/en-us/sql-server/sql-server-downloads)
- [Node.js](https://nodejs.org/) (for Tailwind CSS)
### Installation
 
```bash
# Clone the repository
git clone https://github.com/abdhullah200/Lume.git
 
# Navigate into the project
cd Lume
 
# Install frontend dependencies
npm install
 
# Restore .NET packages
dotnet restore
 
# Apply database migrations
dotnet ef database update
 
# Run the application
dotnet run
```
 
> Make sure to update your connection string in `appsettings.json` before running.
